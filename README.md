# Visor de Trabajos — Claudia Bravo

Dashboard de ofertas laborales (Gestión de Personas · Desarrollo Organizacional · Educación, Santiago de Chile) encontradas por el buscador de trabajo con Claude Code.

**Ver online:** https://langab.github.io/visor_trabajo_claudia/viewer/

- `viewer/` — la página del visor
- `data/jobs.json` — ofertas exportadas desde el scraper

Se actualiza con `python3 tools/update_visor.py --push` desde el repo `ai-job-search`.
